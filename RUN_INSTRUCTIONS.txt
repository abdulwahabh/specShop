
OPTIMASTER PRO - DESKTOP SYSTEM SETUP
=====================================

Follow these steps exactly to fix the login issue and create your desktop icon.

STEP 1: DATABASE INITIALIZATION (CRITICAL)
------------------------------------------
1. Open MySQL Workbench or your MySQL Command Line.
2. Copy all the code from 'migration.sql' and run it.
3. This creates the 'optimaster_db' and the Admin user needed for login.

STEP 2: CONFIGURE DATABASE PASSWORD
-----------------------------------
1. Open 'server.js' in a text editor.
2. Find: password: 'your_password'
3. Replace 'your_password' with your actual MySQL root password.
4. Save the file.

STEP 3: CREATE DESKTOP ICON
---------------------------
1. Go to your project folder.
2. Right-click the file named 'launch_optimaster.bat'.
3. Select 'Send to' -> 'Desktop (create shortcut)'.
4. Go to your Desktop, right-click the shortcut -> Rename to "OptiMaster Pro".
5. (Optional) Right-click shortcut -> Properties -> Change Icon to give it a professional look.

STEP 4: RUNNING THE APP
-----------------------
1. Double-click your "OptiMaster Pro" desktop icon.
2. Wait a few seconds for the servers to start.
3. A clean window will open without browser address bars.
4. LOGIN CREDENTIALS:
   Email: admin@optimaster.com
   Password: admin123

TROUBLESHOOTING LOGIN:
----------------------
- If it still says "Invalid", ensure you ran 'migration.sql' in MySQL.
- Check the console window that opens when you run the .bat file to see if there are database connection errors.
